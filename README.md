# BusLocationTracker (Driver version)
*****
## 아파트 셔틀버스 위치 추적 앱(Bus Location Tracking App)
    아파트 셔틀버스의 위치를 파악하여 셔틀버스 이용을 편리하게 만들어주기 위함.
    버스에 안드로이드 스마트폰을 설치하여, 서버와 Restful 통신을 통해  
    버스가 운행중임을 알리고 위치를 웹에서 출력한다.
