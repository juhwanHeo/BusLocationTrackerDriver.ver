# BusLocationTracker
*****
## 아파트 셔틀버스 위치 추적 앱(Bus Location Tracking App)
    아파트 셔틀버스의 위치를 파악하고 셔틀버스 이용을 편리하게 만들어주기 위함.
    버스 운전 기사님의 휴대폰과 버스 내부에 별도의 장치를 통해 기사님의 휴대폰과 버스에 설치된 장치가 연동하여
    버스가 운행중임을 알리고 위치를 앱에서 실시간으로 표시함
